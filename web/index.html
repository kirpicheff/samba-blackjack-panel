Для работы с доменом на сервере должны быть установлены пакеты:

Присоединение к домену

Для ввода сервера в домен требуются учетные данные администратора домена. Эти данные не сохраняются в системе и используются только один раз.

Сетевое обнаружение

Как это работает?

WSDD позволяет серверу отображаться в разделе «Сеть» (Network) в Windows 10/11.
Avahi обеспечивает обнаружение сервера в macOS (Finder) и Linux через протокол mDNS/Bonjour.
