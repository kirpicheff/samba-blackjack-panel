# Samba Blackjack Panel

Легковесная панель управления сервером Samba на Go.

## Возможности
- Мониторинг активных сессий и открытых файлов.
- Просмотр списка общих ресурсов (shares) из `smb.conf`.
- Управление пользователями Samba.
- Режим разработки (Mock-режим) для запуска на Windows/macOS.

## Запуск
1. Убедитесь, что у вас установлен Go.
2. Склонируйте репозиторий.
3. Запустите проект:
   ```bash
   go run .
   ```
4. Откройте в браузере `http://localhost:8888`.

## Требования и зависимости (Linux)

Для полноценной работы всех функций панели в Linux-системе (рекомендуется Ubuntu/Debian) необходимо установить следующие пакеты:

### 1. Основные (База)
```bash
sudo apt update
sudo apt install samba samba-common-bin
```
*   `smbstatus` — для мониторинга сессий.
*   `pdbedit` — для управления пользователями.

### 2. Интеграция с Active Directory (Этап 5)
```bash
sudo apt install krb5-user winbind libnss-winbind libpam-winbind
```
*   Необходимо для работы команды `net ads join` и авторизации через домен.

### 3. Сетевое обнаружение (Этап 6)
```bash
sudo apt install avahi-daemon
# wsdd обычно ставится из сторонних репозиториев или скачивается скриптом
sudo apt install wsdd 
```
*   `avahi-daemon` — для обнаружения в macOS/Linux (mDNS).
*   `wsdd` — для обнаружения в Windows 10/11 (Web Services Dynamic Discovery).

### 4. Дополнительные модули
*   `samba-vfs-modules` — требуется для работы Аудита (`full_audit`) и Корзины (`recycle`).
*   `acl` — для управления расширенными правами доступа (`getfacl`/`setfacl`).

## Запуск проекта

1.  **Установка зависимостей Go**:
    ```bash
    go mod download
    ```
2.  **Запуск**:
    ```bash
    go run .
    ```
3.  **Доступ**: Панель будет доступна по адресу `http://localhost:8888`.
    *Логин/пароль по умолчанию: `admin / admin` (настраивается в main.go).*

## Режим разработки (Windows/macOS)
Если вы запускаете панель не на Linux, она автоматически переходит в **Mock-режим**:
- Вместо реальных системных вызовов используются заглушки с тестовыми данными.
- Создается локальный файл `smb.conf.dev` для имитации конфигурации.
